# CyberEdge 互联网综合扫描 攻击面测绘

## 0x01 简介

CyberEdge是一款基于已有扫描器进行集成的互联网综合扫描器，可用于网络安全行业进行各种测试。

CyberEdge采用 域名——子域名——IP——端口——目录——指纹——漏洞 结构进行综合扫描。

## 0x05 部署方法

```bash
docker-compose up -d
```

前端位于4567端口

后端位于1234端口

## 0x06 使用的组件

端口扫描: Nmap
